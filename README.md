不会写浏览器脚本故暂且用剪贴版复制传一下信息

`python convert.py`后输入 checklist id 并处理需要特判的鸟种即可, `n/N`表示忽略, `_`表示重复名字 (e.g. 在处理橙腹叶鹎时, `'西南_'` for `'西南橙腹叶鹎'`)

比起用看不懂的学名转换, 用中文名转换更好维护点. 不过我不太清楚有多少中文命名 / 分类学分分合合上的出入, 欢迎 pr/issue 提供信息.
